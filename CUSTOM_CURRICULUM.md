# 나만의 커리큘럼 짜기 — 기능 명세서

## 1. 기능 개요

사용자가 5가지 진로 분야 중 하나를 선택하면, 학과 커리큘럼 전체에서 해당 분야와 연관된 과목들을 **하이라이트**하여 맞춤 학습 경로를 시각적으로 안내하는 인터랙티브 섹션.

---

## 2. UI 위치

| 항목 | 내용 |
|------|------|
| 기존 섹션 | `HeroUI` → 섹션 3 (Future Vision 카드) |
| 교체 후 | `CustomCurriculumSection` 컴포넌트로 대체 |
| 파일 | `components/SeokyeongHero.jsx` |

---

## 3. 사용자 플로우

```
① 페이지 하단 스크롤
        ↓
② "나만의 커리큘럼 짜기" 섹션 등장 (스크롤 애니메이션 진입)
        ↓
③ 5가지 분야 카드 중 하나 클릭하여 선택 (토글)
        ↓
④ "맞춤 커리큘럼 생성하기" 버튼 클릭
        ↓
⑤ 위 커리큘럼 섹션의 과목 칩이 하이라이트/딤 처리
   - 해당 과목: 골드 테두리 + 배경 강조
   - 비해당 과목: 흐리게 (opacity 낮춤 + grayscale)
        ↓
⑥ 학년 탭 자유롭게 전환하며 확인 가능
```

## 7. 하이라이트 스타일 스펙

### 매칭 과목 칩 (highlighted)
```css
border: 2px solid #facc15;
background: rgba(250, 204, 21, 0.12);
box-shadow: 0 0 10px rgba(250, 204, 21, 0.3);
color: #fef08a;
```

### 비매칭 과목 칩 (dimmed)
```css
opacity: 0.25;
filter: grayscale(50%);
```

### 트랙 미선택 시
```
모든 과목: 기존 스타일 유지 (하이라이트 없음)
```

## 8. 컴포넌트 구조

```
SeokyeongHero
 ├── selectedTrack (state: null | '웹개발' | '프론트엔드' | ...)
 │
 ├── CurriculumSection (섹션 2)
 │    └── selectedTrack prop 수신
 │         → 과목 칩 렌더 시 하이라이트/딤 판정
 │
 └── CustomCurriculumSection (섹션 3 — Future Vision 대체)
      └── onTrackSelect(track) 콜백 → selectedTrack 업데이트
```

## 9. CustomCurriculumSection UI 구성

```
┌────────────────────────────────────────────────────┐
│  나만의 커리큘럼 짜기                               │
│  관심 분야를 선택하면 맞춤 커리큘럼을 추천해드려요   │
│                                                    │
│  ┌──────┐ ┌──────────┐ ┌────────┐                 │
│  │웹개발 │ │프론트엔드 │ │ 백엔드 │ ...             │
│  └──────┘ └──────────┘ └────────┘                 │
│                                                    │
│  [ 맞춤 커리큘럼 생성하기  →  ]                    │
└────────────────────────────────────────────────────┘
```

- 분야 카드: 글래스모피즘 + 선택 시 해당 트랙 색상으로 테두리/배경
- 생성 버튼: 분야 미선택 시 비활성(disabled) 처리
- 버튼 클릭 시: 스크롤이 커리큘럼 섹션으로 이동 + 하이라이트 적용

## 10. 구현 순서 (다음 프롬프트)

1. `SeokyeongHero`에 `selectedTrack` state 및 `TRACKS` 상수 추가
2. `CurriculumSection`에 `selectedTrack` prop 추가 → 과목 칩 하이라이트 로직 삽입
3. `HeroUI`의 섹션 3(Future Vision)을 `CustomCurriculumSection`으로 교체
4. `CustomCurriculumSection` 컴포넌트 구현
   - 분야 카드 5개 (선택/해제 토글)
   - "맞춤 커리큘럼 생성하기" 버튼
   - 클릭 → `selectedTrack` 업데이트
5. 커리큘럼 섹션 스크롤 연동 (선택 후 섹션 2로 스크롤)

## 11. 데이터 플로우 요약

```
사용자 클릭 (분야 선택)
    → selectedTrack 업데이트 (SeokyeongHero state)
    → CurriculumSection이 prop으로 수신
    → 과목 렌더 시: course.tags.some(t => TRACKS[selectedTrack].tags.includes(t))
        true  → highlighted 스타일
        false → dimmed 스타일
```
